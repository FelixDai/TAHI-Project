    TEST PROCEDURE

                        
       HA0    NUTX      R1      R2     CN0
        |       |       |       |       |
        |       | <---- |       |       | 1.Router Advertisement
        |       |       |       |       |
        | <---- |       |       |       | 2.Binding Update
        | ----> |       |       |       | 3.Binding Acknowledgement
        |       |       |       |       |
        | ----> |       |       |       | 4.ICMP Echo Request
        | <---- |       |       |       | 5.ICMP Echo Reply
        |       |       |       |       |
        |       |       |       |       | 6.(wait)
        |       |       |       |       |
        | <---- |       |       |       | 7.Binding Update (*1)
        | ----> |       |       |       | 8.Binding Acknowledgement
        |       |       |       |       |
                    
        1. Send Router Advertisement. (R1 -> R1_allnode_multi)
        2. Receive Binding Update. (NUTX -> HA0)
        3. Send Binding Acknowledgement. (HA0 -> NUTX)
            # The value of Refresh Interval is less than BU[2].
        4. Send ICMP Echo Request. (HA0 -> NUTX with Type2 Routing Header)
        5. Receive ICMP Echo Reply. (NUTX -> HA0 with Home Address Option)
        6. (wait)
            # Wait during a maximum of the Refresh Interval in the Binding Acknowledgement[3].
        7. Receive Binding Update. (NUTX -> HA0)
        8. Send Binding Acknowledgement. (HA0 -> NUTX)
                    
    Focus Packet Format:
        3. Binding Acknowledgement:
            IPv6 header
                source          = home agent
                destination     = care-of address
            Routing header (type 2)
                address         = home address
            ESP header in transport mode
            Mobility header
              Binding Acknowledgement
                status          = 0
                lifetime        = (same value as Binding Update[2])
                Binding Refresh Advice option
                    interval    = (less than Binding Update[2])
                    
        7. Binding Update

    JUDGEMENT

     (*1) PASS: HA0 receives Binding Update message before the expiration of the following period,
             - The remaining lifetime of the home registration which set to 
               the Refresh Interval of the Binding Acknowledgement.
