JUDGEMENT

SIP - Generic Judgment (make ACK)
 [generic_make_ACK_for-non2XX.txt]
 
 Generic judgement items for ACK request for non-2xx response
 
 - Request-Line:
     Method: Must be "ACK". #7.1 the present tense.
     Request-URI: Must be the same value of that in the original request. [RFC3261-17-32]
     Request-URI: Must not contain unescaped spaces or control characters. [RFC3261-7-4]
     Request-URI: Must not be enclosed in "<>". [RFC3261-7-4]
 
 - Headers:
 
   * From
       Must exist. [RFC3261-8-1]
       Must be the same value of that in the original request. [RFC3261-17-32]
       addr-spec: Must be the specified SIP-URI as NUT(AoR). [RFC3261-20-22]
       addr-spec: Must be enclosed in "<>" if a comma, semicolon, or question mark is contained. [RFC3261-8-9, RFC3261-12-37]
       tag-param: Must exist. [RFC3261-8-9, RFC3261-12-37]
 
   * Call-ID
       Must exist. [RFC3261-8-1]
       Must be the same value of that in the original request. [RFC3261-17-32]
 
   * To
       Must exist. [RFC3261-8-1]
       Must equal the To header field in the response being acknowledged. [RFC3261-17-33]
       addr-spec: Must be the specified SIP-URI as UA1(AoR). (Excepting REGISTER request.)
       addr-spec: Must be enclosed in "<>" if a comma, semicolon, or question mark is contained. [RFC3261-20-22]
       tag-param: tag-param Must equal that in corresponding response, if present.
 	  
   * Via
       MUST be equal to the top Via header field of the original request. [RFC3261-17-35]
 
   * CSeq
       Must exist. [RFC3261-8-1]
       Must be less than 2**31. [RFC3261-8-16]
       method: Must match that of the request. [RFC3261-8-14]
       sequence number: Must be the same value as was present in the original request.
 
   * Max-Forwards
       Must exist. [RFC3261-8-1, RFC3261-8-17]
 
   * Require
       Must not exist. [RFC3261-8-80,RFC3261-20-8]
 
   * Proxy-Require
       Must not exist. [RFC3261-8-80,RFC3261-20-8]
 
   * Accept
       Must not exist. [RFC3261-8-80]
 
   * Accept-Encoding
       Must not exist. [RFC3261-8-80]
 
   * Accept-Language
       Must not exist. [RFC3261-8-80]
 
   * Alert-Info
       Must not exist. [RFC3261-8-80]
 
   * Allow
       Must not exist. [RFC3261-8-80]
 
   * Expires
       Must not exist. [RFC3261-8-80]
 
   * In-Reply-To
       Must not exist. [RFC3261-8-80]
 
   * Organization
       Must not exist. [RFC3261-8-80]
 
   * Priority
       Must not exist. [RFC3261-8-80]
 
   * Peply-To
       Must not exist. [RFC3261-8-80]
 
   * Server
       Must not exist. [RFC3261-8-80]
 
   * Subject
       Must not exist. [RFC3261-8-80]
 
   * Supported
       Must not exist. [RFC3261-8-80]
 
   * Warning
       Must not exist. [RFC3261-8-80]
