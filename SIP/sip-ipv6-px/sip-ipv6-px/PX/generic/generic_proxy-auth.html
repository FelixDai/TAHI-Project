JUDGEMENT

SIP - Generic Judgment (proxy auth)
 [generic_proxy-auth.txt]
 
 Generic judgement items for SIP authentication challenge (Proxy-Authenticate)
 
 - Headers:
 
    * Proxy-Authenticate
    
    challenge: Must begin with "Digest".  # RFC3261 Sec.25 ABNF
    nonce: Must exist. [RFC3261-22-37, RFC2617 3.2.1]
    realm: Must exist. [RFC2617 3.2.1]
    qop: Must exist. [RFC3261-22-36]
         Must equal "auth" (with quotation marks).
    algorithm: Must equal "MD5",if algorithm parameter exist.
