JUDGEMENT

SIP - Generic Judgment (forward from)
 [generic_forward_from-UA11.txt]
 
  Generic judgement items for SIP message forwarded from UA11
 
  The size of whole request message must be less than of equal to 1300bytes.
 
  - Request-Line:
      method: Must equal as that in the message from UA11.
  - Status-Line:
      Status-Code: Must equal as that in the message from UA11.
 
 
  - Headers:
 
    Must not reorder field values with a common field name. [RFC3261-16-45]
 
    * To
        Must exist.
        addr-spec:  Must equal as that in the message from UA11. 
        tag-param:  Must equal as that in the message from UA11(Must not exist unless original message has any value).
 
    * From
        Must exist.
        addr-spec: Must equal as that in the message from UA11.
        tag-param: Must equal as that in the message from UA11(Must not exist unless original message has any value).
 
    * Call-ID
        Must exist.
        Must equal as that in the message from UA11.
 
    * CSeq
        Must exist.
        seq-no: Must equal as that in the message from UA11
        method: Must match that of the request.
                    Must equal as that in the message from UA11
 
    * Max-Forwards
        Must exist.
        Must be the value specified in the tester configuration.
        Must be decremented by one from that in the message from UA11. [RFC3261-16-49]
 
    * Content-Length
        Must exist.
        Must equal as that in the message from UA11.
 
  - Bodies:
        Must equal as that in the message from PX2(not be added, modified, deleted).  [RFC3261-16-46]
