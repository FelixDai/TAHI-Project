JUDGEMENT

SIP - Generic Judgment (forward response)
 [generic_forward_response.txt]
 
 Generic judgement items for SIP response that was forwarded
 
 - Headers:
 
    * Via
        Must exist.
        The first Via header line must not exist in original message. # 16.6 Item 3 the present tense.
 
        For the Via header that was in original message, 
            via-branch: Must equal as that in the message.
            sent-by: Must equal as that in the message.
            transport: Must equal as that in the message.
 
   * Record-Route
       Must include the Record-Route header value in original message.
