JUDGEMENT

SIP - Generic Judgment (Initial INVITE)
 [generic_Initial-INVITE-Session-Timer.txt]
 
 Generic judgement items for Initial INVITE with Session Timer
 
 - Headers:
 
   * Require
     option-tag: Not recommended to include the 'timer' option tag. [RFC4028-7-3]
 
   * Proxy-Require
     option-tag: Not recommended to include the 'timer' option tag. [RFC4028-7-3]
 
   * Supported
     Must exist. [RFC4028-7-1]
     Must exist, if Require or Proxy-Require header fields are present containing 'timer'. [RFC4028-7-4]
     option-tag: Must include the "timer" option tag. [RFC4028-7-1]
     option-tag: Must include the "timer" option tag, if Require or Proxy-Require header fields are present containing 'timer'. [RFC4028-7-4]
 
   * Session-Expires
     delta-seconds: Should not be less than 1800. [RFC4028-4-3]
     delta-seconds: Recommended to be 1800. [RFC4028-4-1]
     delta-seconds: Must be greater than the value in Min-SE(if Min-SE header is present). [RFC4028-7-5]
     refresher-param: Recommended to be omitted. [RFC4028-7-6]
 
   * Min-SE
     delta-seconds: Must not be less than 90. [RFC4028-5-1]
