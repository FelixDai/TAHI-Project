JUDGEMENT

SIP - Generic Judgment (Initial INVITE)
 [generic_Initial-INVITE.txt]
 
 Generic judgement items for Initial INVITE 
 
 - Request-Line:
     Method: Must be "INVITE". #7.1 the present tense.
     Request-URI: Must be the same value as the addr-spec in the To header. [IG-1-10]
 
 - Headers:
 
   * To
       tag-param: Must not exist. [RFC3261-8-7]
 
   * Contact
       Must exist. [RFC3261-12-20]
       addr-spec: Must be the specified SIP-URI as NUT(Contact Address). [RFC3261-12-20, Lmt-3]
       addr-spec:
         host: Recommended to be specified hostname of NUT. [RFC3261-19-3]
       addr-spec: Must be enclosed in "<>" if a comma, semicolon, or question mark is contained. [RFC3261-20-22]
 
   * Content-Type
       Must exist. [RFC3261-20-31, IG-1-16]
       media-type: Must be "application/sdp". [Lmt-4]
 
   * Allow
       Should exist. [RFC3261-13-2]
 
   * Supported
       Should exist. [RFC3261-13-4]
       (option-tag: Must be corresponding to standards-track RFCs, however the tester does not judge this.) [IG-1-14]
 
 - Bodies:
   Must exist. [IG-1-16]
