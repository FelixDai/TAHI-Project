 Generic judgement items for User-Provided Privacy
 
 - Headers:
 
   Should not include any optional headers that might divulge personal information. [RFC3323-4-1]
 
   * Call-ID
       Should use the IP address or hostname for Call-ID. [RFC3323-4-4]
 
   * From
       Should use a display-name of "Anonymous". [RFC3323-4-5]
       Should use the hostname value "anonymous.invalid" for anonymous URI. [RFC3323-4-6]
