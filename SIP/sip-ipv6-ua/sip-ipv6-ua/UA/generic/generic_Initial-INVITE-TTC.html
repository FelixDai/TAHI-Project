JUDGEMENT

SIP - Generic Judgment (Initial INVITE)
 [generic_Initial-INVITE-TTC.txt]
 
 Generic judgement items for Initial INVITE (be base on TTC)
 
 - Headers:
 
   * Route
     Must not exist. [JJ-90.24-13.3.5]
 
   * Require
     Must not exist. [JJ-90.24-13.3.5]
 
   * Proxy-Require
     Must not exist. [JJ-90.24-13.3.5]
 
   * Content-Length
     Must exist. [JJ-90.24-13.3.5]
 
 - Bodies:
   Must exist. [JJ-90.24-10.2.1]
 
   * "m=" line : Must include only one. [JJ-90.24-10.2.1]
 
   * "a=" line   a=<attribute>, a=<attribute>:<value>
       <attribute>: Must not be "inactive" or "sendonly", "recvonly". [JJ-90.24-10.2.1]
       <value>: The value that attribute is "ptime" recommend setting 20 msec. [JJ-90.24-10.2.1]
