JUDGEMENT

SIP - Generic Judgment (digest auth)
 [generic_digest-auth.txt]
 
 Generic judgement items for Authorization and Proxy-Authorization header 
 
 -Headers: 
 
   * Authorization
   * Proxy-Authorization
 
     credentials: Must begin with "Digest". [IG-1-3,IG-1-6,IG-1-12,IG-1-20]
     username: Must exist. [RFC2617 3.2.2]
     username-value: Must be set to the specified value. [RFC2617 3.2.2]
     realm: Must exist. [RFC2617 3.2.2]
     realm-value: Must equal that of the WWW-Authenticate header in 401 response or the Proxy-Authenticate header in 407 response. [RFC2617]
     nonce: Must exist. [RFC2617 3.2.2]
     nonce-value: Must equal that of the WWW-Authenticate header in 401 response or the Proxy-Authenticate header in 407 response. [RFC2617]
     digest-uri: Must exist. [RFC2617 3.2.2]
     digest-uri-value: Must be enclosed in quotation marks. [RFC3261-22-34]
     digest-uri-value: Must equal Request-URI. [RFC3261 22.4 the present tense, RFC2617 3.2.2]
     message-qop: Must exist. [RFC3261-22-37, RFC2617]
     qop-value: Must be set to "auth". [RFC3261-22-37]
     nonce-count: Must exist. [RFC3261-22-37, RFC2617 3.3.2]
     cnonce: Must exist. [RFC3261-22-37, RFC2617 3.3.2]
     dresponse: Must exist. [RFC2617 3.3.2]
     request-digest: Must be calculated in the manner of RFC2617 3.2.2.[MD5/qop=auth]
