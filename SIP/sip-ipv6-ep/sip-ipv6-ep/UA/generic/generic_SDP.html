 [generic_SDP.txt]
 
 Generic judgement items for SDP 
 
   The lines of v, o, s, c*, t, media-part must appear in this order. (Items marked with a "*" may be omitted. ) [RFC3264-5-1, RFC2327]
   The lines of m, c*, a* must appear in this order in the media part. (Items marked with a "*" may be omitted. ) [RFC3264-5-1, RFC2327]
   If a c line in the session-part is omitted, all media-parts must include a c line. [RFC3264-5-1, RFC2327]
   Only one SDP body Must exist. [RFC3264-5-4, RFC2327]
 
   * "v=" line  v=0
       Must be "0". [RFC3264-5-1, RFC2327]
 
   * "o=" line   o=<username> <session id> <version> <network type> <address type> <address>
       <session id>: Must be a 64 bit signed integer. [RFC3264-5-2,3]
       <version>: Must be a 64 bit signed integer. [RFC3264-5-2,3]
       <network type>: Must be "IN". [RFC3264-5-1, RFC2327]
       <address type>: Must be "IP6". [RFC3264-5-1, RFC2327]
       <address>: Must be specified hostname or global unicast address of NUT. [RFC3264-5-1, RFC2327]
 
   * "s=" line  s=<session name>
       <session name>: Recommended to be a single space character (0x20) or a dash (-). [RFC3264-5-5, Lmt-6]
 
   * "c=" line  c=<network type> <address type> <connection address>
       <network type>: Must be "IN". [RFC3264-5-1, RFC2327]
       <address type>: Must be "IP6". [RFC3264-5-1, RFC2327]
       <connection address>: Must be the global unicast address of NUT. [RFC3264-5-1, RFC2327]
 
   * "t=" line  t=<start time> <stop time>
       Should be "0 0" if the SDP is a offer. [RFC3264-5-6]
       Must equal that of the offer if the SDP is an answer. [RFC3264-6-5]
 
   * "m=" line  m=<media> <port> <transport> <fmt list>
       Must contain the same number of "m=" lines as the offer if the SDP is an answer. [RFC3264-6-3]
 
   * "a=" line   a=<attribute>, a=<attribute>:<value>
       If the attribute is "ptime", the value must be greater than zero. [RFC3264-5-18, RFC3264 6.1 the present tense]
