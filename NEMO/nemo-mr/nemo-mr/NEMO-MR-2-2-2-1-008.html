    TEST PROCEDURE

    
      LFN0    NUTX     HA0      R1      R2     CN0
        |     NUT0      |       |       |       |
        |       |       |       |       |       |
        |       | <---- |       |       |       | 1.Router Advertisement
        |       |       |       |       |       |
        |       | ------------> |       |       | 2.Neighbor Solicitation (NUD)
        |       |       |       |       |       | 3.(no reply)
        |       |       |       |       |       |
        |       | ----> |       |       |       | 4.Binding Update
        |       | <---- |       |       |       | 5.Binding Acknowledgement (status=133)
        |       |       |       |       |       |
        |       |       |       |       |       | :
        |       |       |       |       |       |
        |       | ----> |       |       |       | 6.Neighbor Solicitation (DAD) (*1)
        |       |       |       |       |       | 7.(no reply)
        |       |       |       |       |       |
        1. Send Router Advertisement. (HA0 -> HA0_allnode_multi)
        2. Receive Neighbor Solicitation (NUD). (NUTX -> R1)
        3. (no reply)
            # Wait during a maximum of 3 seconds(RFC2461).
        4. Receive Binding Update. (NUT0 -> HA0)
            # The Mobile Router Flag (R) is set to 1.
            # The Lifetime field is set to 0.
        5. Send Binding Acknowledgement. (HA0 -> NUT0)
            # The Status field is set to 133(Not home agent for this mobile node).
            # The Mobile Router Flag (R) is set to 1.
        6. Receive Neighbor Solicitation (DAD). (Unspecified -> solicited-node multicast)
        7. (no reply)
    Focus Packet Format:
        5. Binding Acknowledgement

        6. Neighbor Solicitation (DAD) Data is:
            IPv6 header
                source          = 0::0
                Destination     = solicited-node multicast
            ICMPv6 header
              Neighbor Solicitation
                code            = 0
                target address  = link-local address of nut0

    JUDGEMENT

          (*1) PASS: HA0 receives Neighbor Solicitation (DAD).
            Then, check whether this packet fills all of the following.
             - The target address is a link-local address.
